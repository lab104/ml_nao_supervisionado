Padroniza e gera os conjuntos de treino, validação e teste¶

Foram criados os seguintes conjuntos de dados

  • X_train / y_train com 64% dos dados. Este conjunto ser usado para o treinamento do modelo.
  • X_validation / y_validation com 16% dos dados. Este conjunto ser usado para a validao do modelo.
  • X_test / y_test com 20% dos dados. Este conjunto ser usado para o teste final do modelo.

Como o conjunto de dados muito desbalanceado, os dados da classe 1 (fraude) foi duplicado no conjunto de dados de treinamento para garantir um bom treinamento da rede neural.

def bloco():

    global model
    
    model = tf.keras.models.Sequential([    
        tf.keras.layers.Dense(32, activation='relu', input_shape=X_train.shape[1:]),
        tf.keras.layers.Dense(32, activation='relu', input_shape=X_train.shape[1:]),
        tf.keras.layers.Dense(1, activation='sigmoid')
    ])

    model.summary()
    
    METRICS = [
      tf.keras.metrics.TruePositives(name='positivo_correto',thresholds=0.5),
      tf.keras.metrics.FalsePositives(name='positivo_incorreto',thresholds=0.5),
      tf.keras.metrics.TrueNegatives(name='negativo_correto',thresholds=0.5),
      tf.keras.metrics.FalseNegatives(name='negativo_incorreto',thresholds=0.5), 
      tf.keras.metrics.BinaryAccuracy(name='acc',threshold=0.5),
      tf.keras.metrics.Precision(name='precision',thresholds=0.5),
      tf.keras.metrics.Recall(name='recall',thresholds=0.5),
      tf.keras.metrics.AUC(name='auc'),
      tf.keras.metrics.AUC(name='prc', curve='PR'), # precision-recall curve
    ]
    
    model.compile(
        optimizer=tf.keras.optimizers.RMSprop(
            learning_rate=0.0001, 
            decay=1e-06
        ),    
        loss=tf.keras.losses.BinaryCrossentropy(),
        metrics=METRICS
    )
    
    print( 'OK, modelo compilado...' )
